# 🧳 Reverse Roster (리버스: 1999 명함 생성기)

**Reverse Roster**는 게임 [리버스: 1999 (Reverse: 1999)](https://reverse1999.bluepoch.com/kr/) 유저들을 위한 비공식 마도학자(캐릭터) 보유 현황 트래커 및 명함 생성 웹 툴입니다. 자신이 보유한 캐릭터를 선택하고, 한 장의 깔끔한 이미지로 저장하여 커뮤니티에 공유하거나 덱 상담을 받을 수 있습니다.

## ✨ 주요 기능 (Features)
- **마도학자 선택 및 관리:** 직관적인 UI를 통해 보유 중인 마도학자 체크
- **필터링 시스템:** 성급(6성, 5성 등) 및 주요 키워드(추가 공격, 중독 등) 기반 필터링
- **명함 이미지 생성:** 선택한 마도학자 목록을 한 장의 이미지로 렌더링 및 다운로드 (`html2canvas` 활용)
- **반응형 웹 디자인:** PC 및 모바일 환경 지원 (예정)

## 🛠 기술 스택 (Tech Stack)
- **Frontend:** React (Vite)
- **Styling:** Vanilla CSS
- **Deployment:** GitHub Pages (예정)
